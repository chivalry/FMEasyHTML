FMEasyHTML: Frequently Asked Questions

What is FMEasyHTML?
FMEasyHTML is a way to provide FileMaker users with a WYSIWYG HTML / text editor, making it easy for them to work on HTML content and see it as it will ultimately be rendered. It's ideal for applications in which the content is being stored in a FileMaker database and will ultimately be published to the Web.

Can FMEasyHTML be used with FileMaker Go and/or FileMaker WebDirect?
FMEasyHTML is currently compatible with FileMaker Pro on the Mac, and FileMaker Go. We're working on support for Windows.

Unfortunately, due to limitations of FileMaker WebDirect, FMEasyHTML does is not yet compatible with it.

Does FMEasyHTML require the use of a plugin?
No. FMEasyHTML uses only native FileMaker functionality. No plugins are needed.

Does FMEasyHTML need an active internet connection in order to work properly?
No. You can use the editor whether you are online or offline.

What technologies does FMEasyHTML use?
FMEasyHTML uses jQuery and jQuery TE. Use of FMEasyHTML is subject to the licenses of each of those technologies. For more information, see the "License" file located in the FMEasyHTML zip file.

How much does FMEasyHTML cost?
FMEasyHTML is being provided via an open source license, and is based on technology that is provided under a similar license.

Who developed FMEasyHTML?
FMEasyHTML was developed by Tim Dietrich, a database consultant that specializes in the FileMaker platform.
